# 예제 코드 (Example Code)

이 디렉토리는 강의에서 사용되는 예제 코드를 포함합니다.

This directory contains example code used in lectures.

## 디렉토리 구조 (Directory Structure)

- `basic/` - 기초 예제 (Basic examples)
- `advanced/` - 심화 예제 (Advanced examples)

## 예제 사용 방법 (How to Use Examples)

1. 관심있는 예제 디렉토리로 이동
2. README.md 파일의 설명 읽기
3. 예제 코드 실행 및 수정
4. 다양한 방식으로 실험해보기

## 예제 목록 (Example List)

### 기초 예제 (Basic Examples)
- Hello World
- 변수와 데이터 타입
- 조건문
- 반복문

### 심화 예제 (Advanced Examples)
- 함수와 모듈
- 데이터 구조
- 파일 입출력
- 에러 처리
