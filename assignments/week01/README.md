# Week 01 과제: 환경 설정 및 기초

## 과제 목표 (Assignment Goals)

개발 환경을 설정하고 기본적인 프로그래밍 개념을 실습합니다.

## 과제 내용 (Assignment Tasks)

### Task 1: 환경 설정 확인 (10점)
- 개발 환경이 올바르게 설정되었는지 확인
- 필수 도구가 모두 설치되었는지 체크리스트 작성

### Task 2: Hello World 프로그램 (30점)
- "Hello, FS11!" 을 출력하는 프로그램 작성
- 코드에 주석 추가
- 프로그램 실행 결과 스크린샷 포함

### Task 3: Git 기초 (30점)
- Git 저장소 생성
- 첫 번째 커밋 작성
- README.md 파일 작성

### Task 4: 자기소개 (30점)
- 간단한 자기소개 프로그램 작성
- 이름, 학습 목표, 관심 분야 출력

## 제출 방법 (Submission)

1. `week01/` 디렉토리에 모든 파일 저장
2. 실행 방법을 README.md에 작성
3. 마감일까지 제출

## 제출 기한 (Deadline)

Week 2 시작 전까지

## 평가 기준 (Rubric)

- 완성도: 각 Task의 요구사항을 모두 충족했는가?
- 코드 품질: 코드가 깔끔하고 이해하기 쉬운가?
- 문서화: 충분한 주석과 설명이 있는가?
