# Week 02 과제: 기초 프로그래밍

## 과제 목표 (Assignment Goals)

변수, 데이터 타입, 제어 구조를 활용하여 프로그램을 작성합니다.

## 과제 내용 (Assignment Tasks)

### Task 1: 계산기 프로그램 (25점)
- 두 숫자를 입력받아 사칙연산을 수행하는 프로그램
- 덧셈, 뺄셈, 곱셈, 나눗셈 기능 구현
- 0으로 나누는 경우 에러 처리

### Task 2: 학점 계산기 (25점)
- 점수를 입력받아 학점을 출력하는 프로그램
- 90점 이상: A, 80점 이상: B, 70점 이상: C, 60점 이상: D, 그 외: F
- 조건문을 사용하여 구현

### Task 3: 구구단 출력 (25점)
- 사용자가 입력한 단의 구구단을 출력
- 반복문을 사용하여 구현
- 1단부터 9단까지 선택 가능

### Task 4: 숫자 맞추기 게임 (25점)
- 1부터 100 사이의 랜덤 숫자를 맞추는 게임
- 사용자 입력과 비교하여 힌트 제공 (더 크다/작다)
- 맞출 때까지 반복

## 제출 방법 (Submission)

1. 각 Task별로 별도의 파일로 작성
2. `week02/` 디렉토리에 모든 파일 저장
3. 실행 방법을 README.md에 작성

## 제출 기한 (Deadline)

Week 3 시작 전까지

## 추가 도전 과제 (Bonus Challenge)

- 사용자 입력 유효성 검사 구현 (+5점)
- 프로그램 UI 개선 (+5점)

## 평가 기준 (Rubric)

- 기능 구현: 모든 요구사항이 정확히 동작하는가?
- 에러 처리: 잘못된 입력에 대한 처리가 있는가?
- 코드 구조: 코드가 읽기 쉽고 유지보수가 가능한가?
- 문서화: 각 함수와 로직에 대한 설명이 충분한가?
