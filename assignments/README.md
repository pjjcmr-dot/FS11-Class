# 과제 (Assignments)

이 디렉토리는 주차별 과제를 포함합니다.

This directory contains weekly assignments.

## 과제 제출 방법 (Submission Guidelines)

1. 각 과제의 요구사항을 꼼꼼히 읽으세요.
2. 제공된 템플릿을 사용하여 과제를 완성하세요.
3. 코드는 명확하게 주석을 작성하세요.
4. 테스트를 통과하는지 확인하세요.

## 평가 기준 (Grading Criteria)

- 요구사항 충족: 40%
- 코드 품질: 30%
- 문서화: 20%
- 창의성: 10%
