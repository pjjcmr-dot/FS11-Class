# Week 02: 기초 개념

## 학습 목표 (Learning Objectives)

1. 프로그래밍 기초 개념 이해
2. 데이터 타입과 변수
3. 제어 구조 학습

## 강의 내용 (Lecture Content)

### 1. 데이터 타입과 변수
- 기본 데이터 타입
- 변수 선언과 사용
- 상수와 리터럴

### 2. 연산자
- 산술 연산자
- 비교 연산자
- 논리 연산자

### 3. 제어 구조
- 조건문 (if, else, switch)
- 반복문 (for, while, do-while)
- 제어문 (break, continue)

## 실습 (Practice)

1. 변수 선언 및 초기화 연습
2. 조건문을 사용한 프로그램 작성
3. 반복문을 활용한 패턴 출력
